akshayrs_Project2.m is the main executable file. 

The functions inCSpace, outObstacle1,outObstacle2 and outObstacle3 are used to check if the point lies inside the configuration
space and is not in contact with any of the obstacles.

The functions move1..move8 are the move actions to the 8 points around the current point.

The function plotPointsOnCspace is used to plot the output path to the goal position after generating the nodes.
